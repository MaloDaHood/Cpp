void entree_securisee ( T &  variable,
Predicat  predicat 
)

Saisie sécurisée avec std::cin.

Vérifie que la variable demandée est du bon type et respecte un prédicat donné. En cas de fermeture du flux d'entrée, une exception std::runtime_error est levée. En cas de type invalide, le flux d'entrée est vidé. En cas de prédicat non respecté, un message d'erreur est affiché sur la sortie standard.

Template Parameters
TLe type de la valeur à récupérer.
PredicatUn prédicat à passer pour personnaliser la vérification.
Parameters
[out]variableUne référence sur la variable de type T à récupérer.
[in]predicatUn prédicat que l'entrée saisie doit respecter.
Precondition
Le type T doit être utilisable avec std::cin.
Le prédicat doit être de la forme bool predicat(T variable).
Postcondition
La variable sera initialisée correctement avec la valeur entrée.
Exceptions
std::runtime_errorSi le flux d'entrée est fermé.
