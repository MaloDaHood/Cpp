# C++
The executable files aren't included.
If you want to run the code you will have to compile it.

***

- ## On Windows
> With MinGW installed

``$ g++ file.cpp -o prog.exe``
or
``$ g++ *.cpp -o prog.exe``


***

- ## On Linux 
> If a Makefile is present

``$ make``

> Else you can do the same as Windows
